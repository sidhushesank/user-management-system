# 🚀 User Management System

A role-based User CRUD dashboard built with modern frontend design and REST API integration.

## 🔹 Features

### 👑 Admin
- Secure admin login
- View all users
- Add new users
- Edit user details
- Delete users
- Role distribution analytics (Chart.js)

### 👤 User
- View users
- Add new users
- Restricted from edit/delete access

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express (REST API)
- Database: MongoDB
- Charts: Chart.js
- Hosting: Vercel (Frontend)

## 📸 Screens Included

- Role Selection Screen
- Admin Login
- Dashboard
- Role Distribution Analytics
